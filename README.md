# Java Names Array Project

This is a simple Java program that takes user input for names and stores them in an array. Then it prints all the names with numbering.

---

## 📌 Features
- Takes number of names from user
- Stores names in an array
- Displays all names in formatted output

---

## 🛠️ Technologies Used
- Java
- Scanner class (for input)
- Arrays
